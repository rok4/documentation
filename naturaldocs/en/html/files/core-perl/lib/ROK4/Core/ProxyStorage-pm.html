ROK4::Core::ProxyStorage

Proxy to manipulate different storage types : object or file, we can copy, reference or remove from one to another.

In SWIFT case, actions required a prior authentication.  The token is requested with the first action.  If an action is unauthorized during de processing, we retry once with a new token

Using

use ROK4::Core::ProxyStorage;
