ROK4::Core::GeoVector

Describes a georeferenced image and enable to know its components.

Using

use ROK4::Core::GeoVector;

# GeoVector object creation
my $objGeoVector = ROK4::Core::GeoVector->new("/home/ign/DATA/XXXXX_YYYYY.dbf");

Attributes

completePathstring - Complete path (/home/ign/DATA/XXXXX_YYYYY.dbf)
filenamestring - Just the image name, with file extension (XXXXX_YYYYY.dbf).
filepathstring - The directory which contain the image (/home/ign/DATA)
srsstring - Projection of file
xmindouble - Bottom left corner X coordinate.
ymindouble - Bottom left corner Y coordinate.
xmaxdouble - Top right corner X coordinate.
ymaxdouble - Top right corner Y coordinate.
tablehash - all informations about vector data
