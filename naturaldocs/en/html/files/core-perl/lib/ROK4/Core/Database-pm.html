ROK4::Core::Database

Allow to request a PostgreSQL database.

Using

use ROK4::Core::Database;

# DatabaseSource object creation
my $objDatabase = ROK4::Core::Database->new( $dbname, $host, $port, $username, $password );

Attributes

idstring - Connection identifier, concatenation of following values, used to caching results.
connectionDBI database - PostgreSQL connection, use to execute requests
current_resultsDBI statement - Use to manage big results
