ROK4::Core::Array

Provides functions to compute the max value or the min index of an array.  Do not instanciate.

Using

my @array = (1,3,5,9);
my $minIndex = ROK4::Core::Array::minArrayIndex(@array); # return 0
my $maxArrayValue = ROK4::Core::Array::maxArrayValue($array); # return 9
