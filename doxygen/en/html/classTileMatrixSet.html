Detailed Description

Handle pyramid of matrix of tiles.

Author
Institut national de l'information géographique et forestière

A TileMatrixSet represent a pyramid of TileMatrix in the same coordinate system.

JSON definition of a TileMatrix :

{
"crs": "EPSG:3857",
"orderedAxes": [
"X",
"Y"
],
"id": "PM",
"tileMatrices": [
{
"id": "0",
"tileWidth": 256,
"scaleDenominator": 559082264.028718,
"matrixWidth": 1,
"cellSize": 156543.033928041,
"matrixHeight": 1,
"tileHeight": 256,
"pointOfOrigin": [
-20037508.3427892,
20037508.3427892
]
}
]
}
