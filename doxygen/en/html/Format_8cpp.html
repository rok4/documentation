Detailed Description

Implement the namespaces Compression, SampleFormat, Photometric, ExtraSample et Format.

Implement the namespace ContextType.

  • SampleFormat : managed sample type accepted by Image classes
  • Compression : enumerate and managed different compressions
  • Format : enumerate and managed different formats
  • Photometric : enumerate and managed different photometrics
  • ExtraSample : enumerate and managed different extra sample types
  • ContextType : managed context type for storage
