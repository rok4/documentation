Exception codes defined by WMS, OWS & HTTP specifications.

Enumerator
OWS_MISSING_PARAMETER_VALUE 

Operation request does not include a parameter value, and this server did not declare a default value for that parameter

OWS_INVALID_PARAMETER_VALUE 

Operation request includes a parameter with an invalid value

OWS_VERSION_NEGOTIATION_FAILED 

List of versions in “AcceptVersions” parameter value, in GetCapabilities operation request, did not include any version supported by this server.

OWS_INVALID_UPDATESEQUENCE 

Value of (optional) updateSequence parameter, in GetCapabilities operation request, is greater parameter than current value of service metadata updateSequence number

OWS_NOAPPLICABLE_CODE 

No other exceptionCode specified by this service and server applies to this exception

WMS_INVALID_FORMAT 

Request contains a Format not offered by the server

WMS_INVALID_CRS 

Request contains a CRS not offered by the server for one or more of the Layers in the request

WMS_LAYER_NOT_DEFINED 

GetMap request is for a Layer not offered by the server, or GetFeatureInfo request is for a Layer not shown on the map

WMS_STYLE_NOT_DEFINED 

Request is for a Layer in a Style not offered by the server

WMS_LAYER_NOT_QUERYABLE 

GetFeatureInfo request is applied to a Layer which is not declared queryable

WMS_INVALID_POINT 

GetFeatureInfo request contains invalid X or Y value

WMS_CURRENT_UPDATESEQUENCE 

Value of (optional) UpdateSequence parameter in GetCapabilities request is equal to current value of service metadata update sequence number

WMS_MISSING_DIMENSION_VALUE 

Request does not include a sample dimension value, and the server did not declare a default value for that dimension

WMS_INVALID_DIMENSION_VALUE

OWS_OPERATION_NOT_SUPORTED
