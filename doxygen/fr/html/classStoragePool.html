inlinestatic

Récupère un contexte de stockage.

Si un contexte existe déjà pour ce nom de contenant, on ne crée pas de nouveau contexte et on retourne celui déjà existant. Le nouveau contexte n'est pas connecté.

Paramètres
[in]typetype de stockage pour lequel on veut créer un contexte
[in]trayNom du contenant pour lequel on veut créer un contexte

Get a context storage

If a context already exists for this tray's name, we don't create a new one and the existing is returned. New context is not connected.

Paramètres
[in]typeStorage Type for which context is created
[in]trayTray's name for which context is created
